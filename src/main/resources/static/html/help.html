Help

Afin de vous aider lors de l'utilisation de notre API, voici les liens disponibles pour gérer le flux RSS:

Liste des articles

  • Format XML

    • URL : /rss22/resume/xml
    • Méthode : GET
    • Affiche la liste des articles stockés

  • Format HTML

    • URL : /rss22/resume/html
    • Méthode : GET
    • Affiche la liste des articles stockés

Détails d'un article

  • Format XML

    • URL : /rss22/resume/xml/{guid} {guid} = valeur numérique de l’identifiant
    • Méthode : GET
    • Affiche le contenu complet de l’article dont l’identifiant est {guid}

  • Format HTML

    • URL : /rss22/html/{guid} {guid} = valeur numérique de l’identifiant
    • Méthode : GET
    • Affiche le contenu complet de l’article dont l’identifiant est {guid}

Ajout d'un article dans la base

  • URL : rss22/insert
  • Méthode : POST
  • Le flux reçu est validé par le schéma XSD de définition rss22 Si le flux est déjà présent, (même titre et date) alors une indication d’erreur est retournée. Si l’opération est réussie, alors le flux est ajouté à la base et sa persistance est assurée.

Suppression d'un article

  • URL : /rss22/delete/{guid} {guid} = valeur numérique de l’identifiant
  • Méthode : DELETE
  • Suppression de l’article dont l’identifiant est {guid} Pour garantir la cohérence, tout flux rss22 vide (c’est à dire ne contenant aucun article) doit être supprimé

Recherche d'articles

  • URL : /rss22/search?...
  • Méthode : GET
  • Recherche parmi les articles ceux correspondant aux critères entrés (titre, date au format YYYY-MM-DD)
  • Retourne le flux xml correspond aux articles satisfaisant les critères entrés. Si aucun article n'est trouvé, seul le statut NONE est retourné
